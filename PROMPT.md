Цель:
Добавить регистрацию и вход пользователей через OAuth-провайдера Google.

Требования:

1. Аутентификация
- Использовать Auth.js
- Настроить OAuth-провайдер Google
- Использовать server-side сессии
- После входа пользователь должен иметь стабильный userId

2. База данных
- Использовать Prisma
- Таблица users должна хранить:
  - id
  - email
  - name
  - image
  - createdAt
- При первом входе пользователь автоматически создаётся в БД

3. Связь с данными
- Все промты должны быть связаны с пользователем через userId
- Приватные промты видит только владелец

4. UI
- Страница /login
- Кнопка:
  - "Войти через Google"
- Если пользователь уже авторизован — редирект в личный кабинет

5. Middleware
- Защитить страницы:
  - /dashboard
  - /my-prompts
- Неавторизованных пользователей редиректить на /login

6. Переменные окружения
- GOOGLE_CLIENT_ID
- GOOGLE_CLIENT_SECRET
- AUTH_SECRET
- DATABASE_URL

7. Результат
- Пользователь может:
  - войти через Google
  - выйти
- У каждого промта есть владелец
- Приватные данные недоступны другим пользователям
